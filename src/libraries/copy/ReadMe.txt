1) If you still don't have Scene Builder then:

a) Follow the link and download the Scene Builder: https://www.oracle.com/java/technologies/javafxscenebuilder-1x-archive-downloads.html
b) Use Scene Builder dmg file from 'libraries'

2) File -> Project Structure (cmd + ';') -> Dependencies -> + -> JARs or directories ->

add next two files (you can find them in package 'libraries'):
fontawsomefx-8.1.jar
jfoenix-1.0.0.jar

3) To open fxml file in SceneBuilder -> right click on .fxml file -> Open in SceneBuilder
